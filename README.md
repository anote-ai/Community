# Anote Community Portal

> A nationwide, community-driven initiative accelerating **human-centered AI** through events, open resources, and real-world collaboration.

---

## 🌐 Overview
The **Anote Community Portal** is the hub for AI builders, researchers, operators, and enthusiasts across the U.S. It brings together people, ideas, and resources to make AI more **accessible, ethical, and impactful**.

---

## 🛠️ Getting Started

This repository contains the `frontend/` directory — a React application.

**Prerequisites:** Node.js 18+

```bash
# 1. Clone the repo
git clone https://github.com/anote-ai/Community.git
cd Community

# 2. Install dependencies
cd frontend && npm install --legacy-peer-deps

# 3. Set up environment variables
cp .env.example .env
# Edit .env and fill in REACT_APP_BACK_END_HOST and REACT_APP_FRONT_END_HOST

# 4. Start the dev server
npm start
```

The app will be available at `http://localhost:3000`.

For full setup details, CI/CD, and contribution guidelines, see [CODEBASE_SETUP.md](./CODEBASE_SETUP.md).

## 🚀 Features

### 📅 Events
- Discover upcoming **meet-ups, hackathons, and workshops**.
- Both **in-person (city-based)** and **virtual** gatherings.

### 👥 Members
- Browse and connect with fellow **AI builders, researchers, and operators**.
- Expand your network and find collaborators.

### 📚 Courses
- Self-paced **learning tracks**.
- Access **open-source tools** and curated **reading lists**.

### 📝 Content
- Technical blogs, **slide decks**, and **demo videos** from community experts.
- Knowledge-sharing to accelerate adoption of **human-centered AI**.

### 🤖 Agents
- Explore the **Community Agents Gallery**.
- Domain-specific agents for **daily use cases**.

### ☕ Coffee Chats
- Schedule **1-on-1 conversations** with Anote alumni.
- Gain insights into breaking into **AI research and industry roles**.

### 💼 Job Referrals
- Get **warm introductions** to hiring managers and recruiters inside top AI organizations.

### 🧑‍💻 Job Board
- Curated list of **openings in applied AI, ML research, and data science**.
- Updated **daily**.

### 📰 News
- Stay informed with the **latest AI news, announcements, and breakthroughs**.
